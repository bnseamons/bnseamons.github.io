Projects

JARL (Just Another Robotic Limb)

JARL is an prosthetic arm I have been developing for a quadriplegic patient. The current model has 5 degrees of freedom and mounts to side of a wheel chair and allows users to perform tasks such as pushing elevator buttons and drinking from a cup. The user operates the arm from an Android app that sends commands to the Arduino controlling our arm.
CAD assembly

ItemEyes

ItemEyes is an app designed to help users keep track of purchase information. It has two main parts. On the POS (point of sale) side, the software I developed saves receipt information in a QR code that can be printed on receipts. On the user side, the web app, designed in JavaScript, HMTL and CSS reads in the information and appends it to a database of purchases. The app was designed while working with our mentor, who is blind, and is aimed to help those who can note see keep track of purchase information The app was first designed for iOS to take advantage of Apple's accessibilites features for the visually impaired. *

Gitlet

Gitlet is a clone of the Git version control system I coded in java to expand my understanding of Git. I implemented many local functions including the following: init, add, commit, rm, log, global-log status, checkout, branch, rm-branch, reset, and merge. I also implemented remote commands such as the following: add-remote, rm-remote, push, fetch, and pull. *

Tablut

Tablut is a strategy game played between two players on a chess board. I used java to implement the game and an AI. The AI used the mini-max algorithm to select a move, and was efficient due to alpha-beta pruning I implemented. *
